Documentación

1. ¿De qué trata la aplicación?

La aplicación sirve para que a partir de pasarle un archivo JSON o escribirlo, pegarlo en el formulario te cree las interfaces para TypeScript automáticamente.
Tenemos algunas opciones como:

  1. Formatear el json del formulario.
  2. Eliminar el archivo o lo que has escrito en el formulario.
  3. Cambiarle el nombre a la interfaz principal.
  4. Copiar las interfaces generadas.
  5. Descargar un archivo con las interfaces generadas.

2. ¿Cómo usar la aplicación?

2.1 A la hora de usar la aplicación tenemos dos formas distintas para que nos genere la interfaz:

Introduciendo un archivo JSON.

Escribiendo o pegando el texto en formato JSON.

2.2 A nuestra derecha se nos generará la interfaz automáticamente.

Cambiar el nombre de la interfaz principal.

Opciones de copiar la interfaz o descargarla.

3. Posibles alertas que pueden salir.

  • Poner un archivo que no es JSON.
  • Escribir en el formulario algo que no sea como un archivo JSON.
  • Al modificar el nombre de la interfaz root
  • Eliminar el contenido.
  • Copiar la interfaz.
